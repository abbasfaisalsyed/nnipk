T4B News Ticker is a flexible and easy to use WordPress plugin that allows you to make horizontal News Ticker with four animation effects.


== Description ==
T4B News Ticker is a flexible and easy to use WordPress plugin that allows you to make horizontal News Ticker with four effects: Fade, Slide, Typing and Scrolling. You can show post titles inside the ticker either by using categories, or by using tags of the latest posts. You can also show news headlines by creating custom texts with custom links. You can add this plugin directly in the theme files by using a function or by using a shortcode.


= Usage =

* Install and activate the plugin. Go to your Dashboard, then navigate to "Settings >> Ticker Settings" to configure the plugin.


= Features =

* Easy to customize
* Four types of animation effect
* Option to control animation speed
* Option to control time between the fades
* Option to choose category or tag or custom text
* Option to enter number of posts to display

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Install and activate the plugin.
3. Go to: Settings >> Ticker Settings to configure.
4. Insert your tickers by copying and pasting the provided shortcode.
